# BLOCK BLAST

**BLOCK BLAST** — это современная веб-игра в стиле неон/киберпанк, где нужно размещать блоки на поле 8×8, собирать линии и набирать максимальный счёт. Игра поддерживает систему пользователей, таблицу лидеров и несколько уровней сложности.

## 🎮 Особенности

- **Классический геймплей** — размещайте блоки на поле, собирайте линии
- **3 уровня сложности**:
  - **Beginner** — чистое поле, умные блоки, частые инструменты
  - **Advanced** — каменные блоки, цепочки, сбалансированная генерация
  - **Hardcore** — тяжёлые препятствия, случайный хаос, редкие инструменты
- **Система инструментов**:
  - 🔨 **Hammer** — удалить один блок
  - 💥 **Bomb** — взорвать область 3×3
  - ⚡ **Laser** — очистить ряд или столбец
  - 🔀 **Shuffle** — перемешать блоки в лотке
  - ⛓️ **Chain Break** — удалить все цепочки
- **Таблица лидеров** — глобальный рейтинг с фильтрацией по сложности
- **4 визуальные темы**:
  - 🌃 **Cyberpunk** — неоновое свечение, сине-фиолетовая гамма
  - ⚙️ **Steampunk** — медные оттенки, механические шестерёнки
  - 🌌 **Cosmos** — звёздное небо, глубокие фиолетовые тона
  - 🐷 **Piggy** — розовый стиль, декоративные свинки
- **Адаптивный дизайн** — игра работает на ПК, планшетах и телефонах
- **Сохранение прогресса** — игра автоматически сохраняется в localStorage
- **Инструменты сохраняются** — полученные инструменты не пропадут после перезагрузки

## 📋 Требования

- **Python**: 3.8 или выше
- **База данных**: SQLite (по умолчанию) или PostgreSQL (для продакшена)
- **Браузер**: современный браузер с поддержкой ES6 (Chrome, Firefox, Edge, Safari)

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/block-bast.git
cd block-bast
```

### 2. Создание виртуального окружения

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Скопируйте файл `.env.example` в `.env`:

```bash
copy .env.example .env    # Windows
cp .env.example .env      # Linux/macOS
```

Откройте `.env` и настройте переменные:

```env
# Flask
FLASK_APP=app:create_app
FLASK_DEBUG=true          # true для разработки, false для продакшена
FLASK_ENV=development

# Секретный ключ (обязательно измените для продакшена!)
SECRET_KEY=ваш_секретный_ключ

# База данных
DATABASE_URL=sqlite:///blockbast.db
```

**Важно!** Для генерации безопасного `SECRET_KEY` выполните:

```python
python -c "import secrets; print(secrets.token_hex(32))"
```

### 5. Инициализация базы данных

Создайте таблицы базы данных:

```bash
flask db upgrade
```

Или, если папка `migrations` ещё не инициализирована:

```bash
# Инициализация миграций (только один раз)
flask db init
flask db migrate -m "Initial schema"
flask db upgrade
```

### 6. Запуск приложения

**Режим разработки:**
```bash
flask run
```

**Продакшен режим (с Gunicorn):**
```bash
gunicorn -w 4 -b 0.0.0.0:5000 "app:create_app()"
```

Откройте браузер и перейдите по адресу:
- **Разработка**: http://127.0.0.1:5000
- **Продакшен**: http://your-server:5000

## 🎯 Игровой процесс

### Управление

- **Перетаскивание** — перетащите блок из лотка на поле
- **Инструменты** — кликните на инструмент в панели TOOLS
- **Смена сложности** — кнопки DIFFICULTY над панелью инструментов
- **Смена темы** — кнопки CYBER/STEAM/COSMOS/OINK в шапке

### Правила

1. Размещайте блоки на поле 8×8
2. Собирайте полные линии по горизонтали или вертикали
3. Линии исчезают, принося очки
4. Собирайте несколько линий одновременно для бонусов
5. Игра заканчивается, когда ни один блок не помещается на поле

### Очки

- **Размещение блока**: 10 очков
- **Собранная линия**: 100 очков
- **Комбо (2 линии)**: 200 очков
- **Комбо (3+ линии)**: 300+ очков
- **Инструменты**: открываются каждые 1000 очков (зависит от сложности)

## 🗄️ База данных

### Модели

**User** — пользователи:
- `id` — уникальный идентификатор
- `username` — имя пользователя (уникальное, до 20 символов)
- `password_hash` — хэш пароля
- `created_at` — дата регистрации

**Score** — лучшие рекорды:
- `id` — уникальный идентификатор
- `user_id` — ссылка на пользователя
- `score` — лучший счёт
- `difficulty` — уровень сложности (beginner/advanced/hardcore)
- `updated_at` — дата последнего обновления

**ScoreHistory** — история всех попыток:
- `id` — уникальный идентификатор
- `user_id` — ссылка на пользователя
- `score` — счёт в попытке
- `difficulty` — уровень сложности
- `created_at` — дата попытки

### Миграции

Для применения новых миграций:

```bash
flask db migrate -m "Описание изменений"
flask db upgrade
```

Для отката миграции:

```bash
flask db downgrade -1
```

## 🐛 Решение проблем

### Ошибка "SECRET_KEY must be set"

Убедитесь, что в `.env` установлен `SECRET_KEY`:

```env
SECRET_KEY=ваш_секретный_ключ
```

### Ошибка базы данных

Переинициализируйте базу данных:

```bash
rm instance/blockbast.db  # Windows: del instance\blockbast.db
flask db upgrade
```

### Проблемы с миграциями

Сбросьте миграции и создайте заново:

```bash
rm -rf migrations/  # Windows: rmdir /s migrations
flask db init
flask db migrate -m "Initial schema"
flask db upgrade
```

### Кэш браузера

Очистите кэш браузера или откройте в режиме инкогнито:
- **Chrome**: Ctrl+Shift+N
- **Firefox**: Ctrl+Shift+P
- **Edge**: Ctrl+Shift+N
